The workshop will be held at the Forum Convention Centre in Barcelona.

Call for papers

Reference resolution algorithms are applicable for the processing of discourse and dialogue and more recently became important tools for textual Question Answering (QA) systems and Summarization systems. The objective of this ACL workshop is to bring together researchers whose work focuses on the resolution of reference in natural language and to create a forum for the presentation and discussion of their current results. We invite submission of papers describing recent novel and challenging work/results in reference resolution. The range of topics to be covered will include, but will not be limited to: 		 
  • new reference resolution algorithms,
  • reference resolution and its contribution to discourse processing,
  • new applications of reference resolution methods,
  • theories and formalisms for reference resolution,
  • empirical methods in reference resolution,
  • multilingual and multi-document reference resolution,
  • evaluation of reference resolution,
  • resources used for reference resolution,
  • reference resolution for factual question answering,
  • reference resolution for summarization,
  • reference resolution for dialogue processing,
  • reference resolution for the identification of discourse relations and discourse structure.
